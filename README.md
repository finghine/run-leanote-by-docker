# run-leanote-by-docker
用 docker 运行 leanote 的脚本

# 依赖及配置说明

* Dockerfile 中需要leanote中二进制安装文件
* 运行需要docker及docker-compose环境，安装见`install_depend.sh`
* 运行`run_docker_compose.sh`后，需要运行`restore_db.sh`来恢复数据
*
